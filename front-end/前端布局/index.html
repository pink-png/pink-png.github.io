前端常见布局

发表于 2021-06-25
  • Yume Shoka
首页 前端开发 css
序号 布局/单位 描述
1 流式布局（Flow Layout） 基于元素在文档流中的位置进行自然排列，元素会根据父容器的大小自动调整位置和大小
2 浮动布局（Float Layout） 使用 float 属性使元素脱离文档流并进行排列，通过设置不同的浮动值可以实现多列布局
3 定位布局（Positioned Layout） 使用 position 属性控制元素的位置，常见属性值包括 relative、absolute 和 fixed
4 弹性布局（Flexbox Layout） 使用 flexbox 模型实现灵活的布局，通过 display: flex 设置容器，提供丰富的属性和方法控制元素尺寸、位置和排列顺序
5 表格布局（Table Layout） 使用 HTML 表格标签 table、tr 和 td 等定义表格结构和样式
6 网格布局（Grid Layout） 二维布局系统，通过划分页面为网格进行布局，使用 grid-template-columns 和 grid-template-rows 定义行列
7 百分比单位（%） 相对于父元素的尺寸进行计算，如设置宽度为 50% 表示元素宽度是父元素宽度的一半
8 rem 单位 相对于根元素（html）的字体大小进行计算，可用于统一调整页面元素大小
9 vw/vh 单位 vw 相对于视口宽度的百分比，vh 相对于视口高度的百分比，用于实现响应式布局
10 响应式布局（Responsive Layout） 根据不同设备屏幕大小和分辨率自适应调整布局，使用媒体查询、弹性布局、网格布局等技术
