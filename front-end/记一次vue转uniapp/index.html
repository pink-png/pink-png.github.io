vue的相关标签怎么转?

div  header main section -> view
span -> text
ul li -> view ?  这个还不太确定，这个只要有样式类的情况下直接改
img -> image   这里注意的是转成uniapp的image标签要把宽高都设置一下  
    
总结为只要有class类的情况下，基本上都是用的view

字体处理： px 转 rpx 2倍化怎么处理？ 全局样式怎么转化?

uniapp有个<page-meta :root-font-size="'12px'"/>
1rem 就是12px

现在某个页面盒子得大小是22px  但是实际我需要得大小是44px  
so  在不改变前面数字得情况下，

1rem = 2px

但是这个是有一定缺陷的，浏览器里最小的字体是12px ,如果字体大小已经写死的情况下就不能用了   
so  其实还有一个办法，直接全局替换px 为 px * 2

对于已经使用axios写得api接口，怎么快速转化成uniapp的请求方式

针对当前的项目，进行对函数导出和数据传入进行分布封装

import config from "@/config/index.js"
import store from '@/store/index.js';

export const requestApi = (options) => {

    const ppsessid = store.state.user.ppSessId
    const sessid = store.state.user.sessId
    const cid = store.state.env.cid

    const publicHeaders = {
        "Content-Type": "application/json",
        'Accept': 'application/json',
        'X-Yf-CID': cid,
        'X-Yf-PPSessid' : ppsessid,
        'X-Yf-Sessid' : sessid
    }
    return (function() {
        let {
            url,
            method,
            params,
            data
        } = options
        // console.log('options', options)
        return new Promise((resolve, reject) => {
            uni.request({
                url: config.url.devServer + url,
                method: method || 'GET',
                header: publicHeaders,
                data: data || params,
                success: (res) => {
                    resolve(res);
                },
                fail: (err) => {
                    reject(err);
                }
            })
        });
    })(options)
}

vue中的good-storage unaiapp的替代方案

这个js库其实就是一个操作缓存的库

自己封装一个简单的对象即可解决

let storage = {}
storage.get = function(options) {
    uni.getStorageSync(options)
}
storage.set = function(options) {
    uni.setStorageSync(options)
}
storage.remove = function(options) {
    uni.removeStorageSync(options)
}
